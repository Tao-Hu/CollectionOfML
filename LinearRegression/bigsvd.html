SVD for matrix with large \(n\) and \(p\)

Say \(X\) is an \(n\times p\) matrices with both \(n\) and \(p\) large. We want the first two eigenvectors of \(X'X\), which proportional to the \(p\times p\) sample covariance matrix of \(X\) (assuming \(X\) is centered).

SVD scales like the minimum of \(n\) and \(p\), so if one of them is small SVD is fast. If both are huge we need to do something other than brute force. Subsampling is a common way to approximate the eigenvectors. Here we take a subset of the \(n\) observations to compute the SVD. This can be repeated several times for improved precision.

Generate data

 p <- 1000
 n <- 2000

 t <- seq(0,2*pi,length=p)
 X <- matrix(0,n,p)

 for(i in 1:n){
  X[i,] <- 5 + rnorm(1)*sin(t) + 0.5*rnorm(1)*cos(2*t) + rnorm(p)/5
 }

 X <- sweep(X,2,colMeans(X),"-")

Compute the full SVD

 tick <- proc.time()[3]
 SVD <- svd(X)$v[,1:2]
 tock <- proc.time()[3]
 tock-tick

## elapsed 
##    5.41

 matplot(SVD,type="l")

Compute the approximate SVD by taking \(s\) subsamples of size \(m\)

s <- 10
m <- 50

tick <- proc.time()[3]

sub  <- rank(runif(n))<=m
aSVD <- svd(X[sub,])$v[,1:2]/s

for(j in 2:s){
 sub     <- rank(runif(n))<=m
 ev2     <- svd(X[sub,])$v[,1:2]

 ev2[,1] <- sign(cor(ev2[,1],SVD[,1]))*ev2[,1]
 ev2[,2] <- sign(cor(ev2[,2],SVD[,2]))*ev2[,2]

 aSVD    <- aSVD+ev2/s
}
tock <- proc.time()[3]
tock-tick

## elapsed 
##     0.1

matplot(SVD,type="l",col=1)
matplot(aSVD,type="l",add=TRUE,col=2)
legend("topleft",c("True","Approx"),col=1:2,lwd=1,inset=0.05)
